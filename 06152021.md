# 333. 最大 BST 子树
中等难度，写了一个很垃圾的写法，不想改了。拆成了三个小的部分写的。
> 链接：https://leetcode-cn.com/problems/largest-bst-subtree

```
给定一个二叉树，找到其中最大的二叉搜索树（BST）子树，并返回该子树的大小。其中，最大指的是子树节点数最多的。

二叉搜索树（BST）中的所有节点都具备以下属性：

左子树的值小于其父（根）节点的值。

右子树的值大于其父（根）节点的值。

```
```py
    def largestBSTSubtree(self, root: TreeNode) -> int:
        def checkBST(root,l,r):
            res=[]
            def traverse(root):
                nonlocal res
                if not root:return
                traverse(root.left)
                res.append(root.val)
                traverse(root.right)
                return
            traverse(root)
            tm=res[::]
            tm=list(set(tm))
            tm.sort()
            return tm == res
        def getchild(root):
            if not root:return 0
            child = 1
            left=getchild(root.left)
            right=getchild(root.right)
            return child+left+right
        res=0
        def traverse(root):
            nonlocal res
            if not root:return
            if checkBST(root,99999,-99999):
                res=max(res,getchild(root))
            if root.left:
                traverse(root.left)
            if root.right:
                traverse(root.right)
        traverse(root)
```

# 449. 序列化和反序列化二叉搜索树
说实话一开始就想到了根据前序序列化然后排序后得到中序，但是如果没有的话应该怎么办，普遍二叉树那题怎么做。。。。
> 链接：https://leetcode-cn.com/problems/serialize-and-deserialize-bst
```
序列化是将数据结构或对象转换为一系列位的过程，以便它可以存储在文件或内存缓冲区中，或通过网络连接链路传输，以便稍后在同一个或另一个计算机环境中重建。

设计一个算法来序列化和反序列化 二叉搜索树 。 对序列化/反序列化算法的工作方式没有限制。 您只需确保二叉搜索树可以序列化为字符串，并且可以将该字符串反序列化为最初的二叉搜索树。

```
```py
class Codec:

    def serialize(self, root: TreeNode) -> str:
        """Encodes a tree to a single string.
        """
        if not root:return ""
        return str(root.val) + ","+self.serialize(root.left)+","+self.serialize(root.right)

    def deserialize(self, data: str) -> TreeNode:
        """Decodes your encoded data to tree.
        """
        data=data.split(",")
        preorder=[]
        for i in data:
            if i!="":
                preorder.append(int(i))
        # print(preorder)
        inorder=preorder[::]
        inorder.sort()
        # print(inorder)
        def buildtree(preorder,inorder):
            if not preorder:return None
            if len(preorder)==1:
                return TreeNode(preorder[0])
            root = TreeNode(preorder[0])
            def findindex(k,tmp):
                for i in range(len(tmp)):
                    if tmp[i]==k:
                        return i
            index=findindex(root.val,inorder)
            # print(preorder[1:index+1],inorder[:index])
            # print(preorder[index+1:],inorder[index+1:])
            root.left=buildtree(preorder[1:index+1],inorder[:index])
            root.right=buildtree(preorder[index+1:],inorder[index+1:])
            return root
        return buildtree(preorder,inorder)
```
# 510. 二叉搜索树中的中序后继 II

```
给定一棵二叉搜索树和其中的一个节点 node ，找到该节点在树中的中序后继。如果节点没有中序后继，请返回 null 。

一个节点 node 的中序后继是键值比 node.val 大所有的节点中键值最小的那个。

你可以直接访问结点，但无法直接访问树。每个节点都会有其父节点的引用。节点 Node 定义如下：

class Node {
    public int val;
    public Node left;
    public Node right;
    public Node parent;
}

```
```py
    def inorderSuccessor(self, node: 'Node') -> 'Node':
        root = node
        while root.parent:
            root=root.parent
        found = False
        res=None
        def traverse(root,target):
            nonlocal found
            nonlocal res
            if not root:
                return
            traverse(root.left,target)
            if root.val == target.val:
                found= True
            elif found and not res:
                res = root 
            traverse(root.right,target)
        traverse(root,node)
        return res
```
